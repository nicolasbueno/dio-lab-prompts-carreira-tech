# Projeto DIO — Prompts para Planejamento de Carreira em Tecnologia

Este repositório foi desenvolvido como parte de um desafio da DIO, utilizando prompts de Inteligência Artificial para simular uma entrevista de carreira e gerar um plano de desenvolvimento profissional personalizado.

O projeto utiliza dois agentes baseados em prompts:

- **Agent 1 — Entrevistador de Carreira em Tecnologia**
- **Agent 2 — Planejador de Carreiras**

A proposta foi aplicar os prompts ao meu próprio contexto profissional, registrar as respostas, analisar os caminhos sugeridos e estruturar um plano de carreira voltado para a área de Dados, BI e Analytics.

---

## Objetivo do Projeto

O objetivo deste projeto é demonstrar como prompts bem estruturados podem apoiar decisões de carreira em tecnologia.

A partir de uma entrevista guiada, o primeiro agente identifica interesses, experiências, disponibilidade de estudo e objetivos profissionais. Em seguida, o segundo agente transforma essas informações em um plano prático de evolução, com roadmap de estudos, mapa de habilidades, projeto de portfólio e preparação para entrevistas.

---

## Contexto Utilizado

Para este projeto, utilizei meu próprio perfil profissional como base.

### Perfil resumido

- Experiência em tecnologia desde 2015.
- Atuação com dados, BI, automação, cloud e observabilidade.
- Experiência com Power BI, SQL Server, Python, Azure, Zabbix e monitoramento.
- Vivência em ambientes críticos, incluindo operações, incidentes e indicadores.
- Interesse em transição e crescimento para áreas de Dados, BI, Analytics, Inteligência Artificial e Cloud.

---

## Como o Projeto Foi Estruturado

O projeto foi dividido em quatro partes principais:

1. **Prompts**
   - Armazenamento dos prompts utilizados no desafio.
   - Separação entre Agent 1 e Agent 2.

2. **Entrevista**
   - Registro das respostas fornecidas ao Agent 1.
   - Identificação dos principais interesses, experiências e objetivos.

3. **Análise de Carreira**
   - Ranking das carreiras mais aderentes ao perfil.
   - Escolha da carreira principal para aprofundamento.

4. **Plano de Carreira**
   - Roadmap de 90 dias.
   - Mapa de skills.
   - Projeto de portfólio.
   - Roteiro de entrevistas.
   - Sugestões de trilhas de estudo.
