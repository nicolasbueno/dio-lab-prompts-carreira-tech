🚀 PROJETO DE PORTFÓLIO

PROJETO: **Painel Inteligente de Observabilidade e Custos em Cloud**

O QUE FAZER:
Criar um projeto que simule um ambiente operacional com dados de alertas, incidentes, custos cloud e disponibilidade. A ideia é montar um pipeline em Python para tratar os dados, armazenar a base final e construir um dashboard em Power BI com indicadores de operação, custo, criticidade e tendência. Como diferencial, incluir uma análise simples com IA/ML para classificar alertas por prioridade ou identificar padrões de reincidência.

ENTREGÁVEIS:
- Pipeline Python com tratamento dos dados, validações, logs e geração de base final
- Dashboard Power BI com KPIs operacionais, visão executiva e filtros por período, criticidade e serviço
- Repositório GitHub com README explicando problema, arquitetura, etapas, prints e principais insights

CRITÉRIOS DE ACEITAÇÃO:
- O projeto precisa consumir pelo menos 2 fontes de dados diferentes
- O dashboard precisa responder perguntas de negócio, não apenas mostrar gráficos
- O README precisa explicar claramente o problema, a solução, o impacto e os aprendizados

DICA: use um cenário próximo da sua realidade: alertas, incidentes, disponibilidade, tempo de reação, custos cloud ou volumetria transacional. Isso deixa o projeto mais forte porque conecta dados com operação real.
